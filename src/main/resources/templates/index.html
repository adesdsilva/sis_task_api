Documentação da API

Java Spring Boot + Thymeleaf

Task Routes
Cadastrar Tarefa Method POST

POST Route http://localhost:8080/tasks/{listId}

JSON { "title": "Reunião de trabalho", "description": "Discussão sobre o novo projeto", "dueDate": "29-08-2024", "status": "PENDING" }
Obter Tarefas por Lista Method GET

GET Route http://localhost:8080/tasks/list/{listId}?page=0

Atualizar Tarefa Method PUT

PUT Route http://localhost:8080/tasks/{taskId}

JSON { "title": "Reunião de equipe", "description": "Reunião sobre o progresso do projeto", "dueDate": "30-08-2024", "status": "PENDING" }
Excluir Tarefa Method DELETE

DELETE Route http://localhost:8080/tasks/{taskId}

Concluir Tarefa Method PATCH

PATCH Route http://localhost:8080/tasks/{taskId}/complete

Filtrar Tarefas Method GET

GET Route http://localhost:8080/tasks/filter?completed=true&favorite=false

Filtrar Tarefas Concluídas Method GET

GET Route http://localhost:8080/tasks/completed?completed=true

Filtrar Tarefas Pendentes Method GET

GET Route http://localhost:8080/tasks/pending?pending=true

Filtrar Tarefas Favoritas Method GET

GET Route http://localhost:8080/tasks/favorite?favorite=true

Excluir Tarefas Concluídas da Lista Method DELETE

DELETE Route http://localhost:8080/tasks/lists/{taskListId}/completed

Atualizar Status da Tarefa Method PATCH

PATCH Route http://localhost:8080/tasks/{taskId}/status

JSON { "status": "COMPLETED" }
Favoritar/Desfavoritar Tarefa Method PATCH

PATCH Route http://localhost:8080/tasks/{taskId}/favorite

JSON { "isFavorite": true }
Task List Routes
Cadastrar Lista de Tarefas Method POST

POST Route http://localhost:8080/task-lists

JSON { "title": "Lista de Projetos" }
Obter Listas de Tarefas Ordenadas Method GET

GET Route http://localhost:8080/task-lists/ordered?page=0&size=5

Obter Lista de Tarefas por ID Method GET

GET Route http://localhost:8080/task-lists/{id}

GET Route http://localhost:8080/subtasks/

Criar Subtarefa Method POST

POST Route http://localhost:8080/subtasks
